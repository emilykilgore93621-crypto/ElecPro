# **App Name**: WattsUp Electrical Toolkit

## Core Features:

- User Authentication: Secure user login and passkey system for personalized access.
- Calculation Tools: Integrated Ohm's Law, wire sizing, and box fill calculators for accurate estimations.
- Installation guides: common household device installations - outlets, switches, GFCI, weather- tight, cameras, stoves, dishwasher, garbage disposal, hot water heater oron demand
- Reference Library: NEC code snippets, PGE green book, and NFPA articles with clickable links for quick access to standards and guidelines. LLM tool assistance to navigate all external code documentations to generate helpful guidance.
- Interactive Canvas: Canvas for simulating wiring setups and circuit designs.
- Save & Share: Options to save user configurations for further use and ability to export and print the design for documentation.
- Safety Overview: A catalog that compiles equipment overviews that promotes safe handling and best use.
- Inspection Tips: Guidance on what inspectors typically look for to help ensure compliance.

## Style Guidelines:

- Primary color: Deep blue (#1E3A8A), evoking a sense of trust and reliability, reflecting the app's utility as an electrical reference.
- Background color: Light gray (#F5F5F5), providing a neutral and clean backdrop for the app's interface.
- Accent color: Electric Yellow (#D4FF00), highlighting interactive elements and important information.
- Body font: 'PT Sans', a versatile humanist sans-serif that balances modern aesthetics with a touch of warmth and readability.
- Headline font: 'Space Grotesk', a contemporary sans-serif suitable for headlines
- Code font: 'Source Code Pro' for displaying code snippets.
- Use clear, standardized icons to represent electrical components and actions.
- Implement a tab-based navigation system for easy access to different sections.
- Use subtle animations to provide feedback on user interactions and enhance the user experience.
